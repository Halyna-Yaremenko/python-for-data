# Аналітичний звіт за результатами роботи

## 1. Основна логіка та мета
Код завантажує набір даних про фільми, очищує пропуски в ключових стовпцях, трансформує колонку з жанрами у зручний формат та виконує їхній підрахунок.  

## 2. Дані, які обробляються
Використовується датасет **`movies_metadata.csv`**, що містить інформацію про фільми: опис, жанри, теги, посилання на домашні сторінки, колекції тощо.  
Особлива увага приділяється таким полям:
- *tagline*
- *homepage*
- *belongs_to_collection*
- *genres*

## 3. Обробка даних
1. Заповнення пропусків у текстових змінних:
   - *tagline* → `"without tagline"`,
   - *homepage* → `"no homepage"`,
   - *belongs_to_collection* → порожній словник у вигляді `"{}"`.

2. Видалення всіх рядків, у яких залишились будь-які пропуски.

3. Перетворення колонки *genres*, яка спочатку містить рядки зі словниками, у список назв жанрів.

4. Розгортання списків жанрів у один стовпець та підрахунок частоти кожного жанру.

## 4. Отримані результати
Результатом є частотний розподіл жанрів. Згідно з обчисленням:

- Визначено всі унікальні жанри.
- Пораховано, скільки разів кожен жанр трапляється у датасеті.
- Отримано рейтинг жанрів за популярністю (найчастіше представлені жанри згруповані у спадному порядку).

## 5. Візуалізація
Побудовано **стовпчикову діаграму**, яка відображає:
- по осі X — назви жанрів,
- по осі Y — кількість фільмів цього жанру.

## Підсумок
Код виконує повний цикл первинного аналізу жанрів фільмів: очищення, трансформація, агрегація й візуалізація. 

Результат — найпоширеніші жанри у датасеті, ті у яких зустрічається - Drama.
